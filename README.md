# OneVault Backend

The backend system for OneVault application.

## Getting Started

### Prerequisites

- Node.js
- PostgreSQL (via Prisma)
- NestJS CLI

### Installation

```bash
npm install
```

### Running the application

```bash
# development
npm run dev
```

## Features

- Dynamic API endpoints
- Prisma ORM integration
- Authentication & Authorization
- Document & Vault Management
